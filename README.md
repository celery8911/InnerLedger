# InnerLedger / 长了么？

AI 理解你，区块链记住你。  
这是一个把冥想与觉察「记录为过程」的成长账本：不评判、不指导，只帮你看见变化，并将每一次真实发生过的内在片刻写入链上。

## 为什么特别

- 记录过程而非结果：每次冥想与反思都被见证
- 数据归你所有：原文不上链，隐私留在本地
- 成长可验证：链上时间戳 + Growth SBT 里程碑
- 高效可行：依托 Monad 低成本高频记录

## 你能体验到

- 5 分钟呼吸冥想 + 觉察输入
- AI 温和复述与陪伴式提问
- 成长时间轴与链上证明链接
- 首次记录触发 Growth SBT

## 技术栈概览

- Next.js + TypeScript + RainbowKit/wagmi
- Solidity 合约（InnerLedger + Growth SBT）
- OpenAI LLM 理解层
- Monad Testnet 部署

如果你在寻找一个既温柔又可信的成长记录方式，欢迎加入「长了么？」。
